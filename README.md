# Miniscript: Guía para la Línea de Comandos de Bitcoin Core

Guia para realizar transacciones complejas con Miniscript en linea de comandos con Bitcoin Core vs 30.0.

DISCLAIMER: Esta guia ha sido realizada con fines educativos. Siempre pruebe los comandos en redes de prueba y asegurese que va a poder desbloquear los BTC.

## Requisitos

Entorno linux, Bitcoin Core 30.0, Json

Conocimiento previo de la terminal de linux. Recomendado leer el libro o haber realizado el curso de Bitcoin desde la linea de comandos:

https://github.com/BlockchainCommons/Learning-Bitcoin-from-the-Command-Line/blob/spanish-translation/es/README.md
